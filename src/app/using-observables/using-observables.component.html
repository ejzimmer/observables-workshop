Let's start small

Ok, so first up, let's just get an Observable to print its results to console.log

Have a look at using-observables.component.ts

This line myObservable = of('good', 'news', 'everyone'); is creating an Observable that returns three values.

Your mission, should you choose to accept, is to subscribe to the Observable and print the three values to the console. You can do this in the constructor or the ngOnInit or whatever. We're not writing production code here. 

ngOnInit() {
  this.myObserver.subscribe(result => console.log(result));
}

Do it all again

Now, just indulge me for a moment and subscribe to the Observable again. Like literally just make another copy of the code you just wrote, and check out what happens.

It prints the same thing to the console again! That might not surprise you, but it's important to remember that an Observable isn't an EventStream or anything like that. Every time you subscribe to it, it's like calling a function again.
